# ML-SUT-Course
This repository contains my solutions, code, and notes for the **Machine Learning course** at **Sharif University of Technology**. It includes implementations of homework assignments, the final project, exam materials, and lecture slides.


## 📚 Contents

| Folder | Description |
|--------|-------------|
| **HWs** | Jupyter notebooks and Python scripts solving weekly ML problems (regression, classification, clustering, neural networks, transformers, etc.) |
| **Project** | End-to-end machine learning project |
| **Exams** | Exams taken during the course |
| **Slides** | Instructor's slides covering theory and algorithms |
| **TA** | Supplementary materials from TA sessions (more problem-solving) |
